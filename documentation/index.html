You will need the following sofwares to customize this template.

You onle need to change date in Countdown and the position on the Map.

Countdown:

  1. Open the file main.js
  2. Find the section "Countdown"
  3. In the bottom of this section you'll find the line with the date. This is the date for wedding. 
    						// set your wedding date here
						var deadline = 'July 30 2016';
  4. Change this date to which you want.
  5. Be careful! Change only the date in the quotes in this format 'Month day year'.

Map:

We use google map in this template.
  1. Open the file main.js
  2. Find the section Map. 
    						/* 7. Map
						=========================================*/
						var map;
						function initMap() {
							map = new google.maps.Map(document.getElementById('map_canvas'), {
								center: {lat: 40.89, lng: -73.98},
								zoom: 13,
								scrollwheel: false,
								mapTypeControl: false
							});
							var image = 'img/point.png';
							var beachMarker = new google.maps.Marker({
								position: {lat: 40.89, lng: -73.98},
								map: map,
								icon: image
							});
						}
  3. In this section you need to change coordinates. You can see them in lines. 
    						...
						center: {lat: 40.89, lng: -73.98},
						...
						position: {lat: 40.89, lng: -73.98},
						...
    "center" is the map's center coordinates and "position" is the pointer's position.
  4. You can find your place coordinates using this website http://www.mapcoordinates.net/en
  5. Copy lat and lng of your place from this site and paste them in these lines instead those present.

PHP guide #back to top

In this template we use only one PHP file is the form.php. It's needed for sending emails from attending form to you. This form is fully working, you only need to change settings.

  1. Open the file form.php in the main folder.
  2. Find line with email in the top of the code.
  3. Change "your-email@gmail.com" with your email address.

That's all! Now when your visitor will send the form, you get it on your email.
